rl

A minimal, local-first read later CLI

GitHub Install

Features

Local-first SQLite storage
Fast Minimal deps
Portable JSON export
Search Full-text FTS5
Simple Clean CLI
Private No sync/account

Installation

Go Install

go install github.com/bunchhieng/rl/cmd/rl@latest

Build from Source

git clone https://github.com/bunchhieng/rl.git
cd rl
go build -o bin/rl ./cmd/rl

Quick Start

rl add https://example.com --title "Example" --tags "web"
rl list
rl open <id>
rl done <id>
rl search "query"

Commands

rl add <url> [--title "..."] [--note "..."] [--tags "t1,t2"]
rl list [--unread|--read|--all] [--tag t] [--limit N]
rl open <id>
rl done <id>
rl undo <id>
rl rm <id> [id...]
rl export
rl import <file.json>
rl search "<query>"
rl version

Examples

Add with metadata 
rl add https://golang.org --title "Go" --tags "programming"
rl add https://rust-lang.org --note "Check later"
Filter & export 
rl list --tag programming --limit 10
rl export > backup.json

Database

  • macOS: ~/Library/Application Support/rl/links.db
  • Linux: ~/.config/rl/links.db

Override with --db-path flag.
