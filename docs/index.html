rl

A minimal, local-first read later CLI

GitHub Install

Features

Local-first All data stored in a single SQLite file
Fast Minimal dependencies, quick startup
Portable Easy export/import via JSON
Search Full-text search across all fields
Simple Clean CLI interface
Private No account, no sync, no daemon

Installation

Go Install

go install github.com/bunchhieng/rl/cmd/rl@latest

Download

Download from GitHub Releases

Quick Start

Add a link

rl add https://example.com --title "Example" --tags "web"

List links

rl list

Open a link

rl open <id>

Mark as read

rl done <id>

Search

rl search "query"

Examples

Add links with metadata 
rl add https://golang.org --title "Go" --tags "programming"
rl add https://rust-lang.org --note "Check later"
Filter and list 
rl list --unread
rl list --tag programming
rl list --limit 10
Export and import 
rl export > backup.json
rl import backup.json

Commands

rl add <url> [--title "..."] [--note "..."] [--tags "t1,t2"]
rl list [--unread|--read|--all] [--tag t] [--limit N]
rl open <id>
rl done <id>
rl undo <id>
rl rm <id>
rl export
rl import <file.json>
rl search "<query>"
rl version

Database

Links are stored locally:

  • macOS: ~/Library/Application Support/rl/links.db
  • Linux: ~/.config/rl/links.db

Override with --db-path flag.
